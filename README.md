# University-Admission-Dashboard-
Developed an interactive Power BI dashboard to analyze and compare universities based on placement rates, fees, rankings, and facilities. Enabled data-driven decision-making by visualizing state-wise performance, top institutions, and key admission insights.
## Project Overview
The objective of this project is to analyze and compare university admission insights across different states using an interactive Power BI dashboard. It aims to help students and stakeholders evaluate universities based on placement rates, fee structure, rankings, and facilities for informed decision-making.
## Dataset used
The dataset used in this project contains information about various universities across multiple states.
https://github.com/Aartichoudhary01/University-Admission-Dashboard-/blob/main/india_universities_dataset_2025_2026.xlsx

Key attributes include:
- University Name
- State
- Placement Rate (%)
- Highest Package (LPA)
- Average Fees per Year (INR)
- NAAC & NBA Accreditation
- NIRF Ranking 2025
- Programs Offered
- Facilities
- Companies Visited

  ## The dashboard answers the following key questions:

- What is the maximum placement rate across universities?
- Which university offers the highest placement package?
- What is the average annual fee across universities?
- Which universities rank highest based on placement performance?
- How do placement rates vary across different states?
- What are the top universities in terms of packages and placements?
- Which programs and facilities are offered by universities?

## Process
The project was completed through the following steps:

1. Data Cleaning:
   - Handled missing values and inconsistencies
   - Standardized naming conventions

2. Data Transformation:
   - Created calculated measures (Max Placement Rate, Avg Fees, Highest Package)
   - Categorized universities by state and programs

3. Data Modeling:
   - Structured the dataset for efficient filtering and slicing

4. Dashboard Design:
   - Designed a user-friendly and visually appealing layout
   - Implemented slicers for State, University, and Programs

5. Visualization:
   - KPI cards for key metrics
   - Bar charts for university comparisons
   - Donut charts for state-wise analysis
   - Interactive tables for companies visited and facilities
## Project Insights
- Universities in Rajasthan show the highest placement packages compared to other states.
- Placement rates vary significantly across universities, with top institutions reaching up to 95%.
- Average fees differ widely, indicating variation in affordability among universities.
- Certain universities consistently perform well in both placements and rankings.
- Availability of facilities and company visits plays a key role in placement outcomes.

## Final Conclusion
The University Admission Dashboard provides a comprehensive comparison of universities based on placement performance, cost, and infrastructure. It helps students make informed decisions by identifying top-performing institutions and evaluating trade-offs between fees and placement opportunities.

This dashboard demonstrates how data visualization can simplify complex educational data into actionable insights.
